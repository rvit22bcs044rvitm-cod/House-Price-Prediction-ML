# House-Price-Prediction-ML
Machine Learning project for predicting house prices using regression model

## Project Overview
This project predicts house prices using machine learning regression models based on features like:
- Square Footage
- Bedrooms
- Bathrooms
- Year Built
- Lot Size
- Garage Size
- Neighborhood Quality

## Techniques Used
- Data Preprocessing
- Outlier Removal (IQR)
- Log Transformation
- Feature Scaling
- Linear Regression
- KNN Regression

##  Evaluation Metrics
- MAE
- MSE
- RMSE
- MAPE
- R² Score
- Adjusted R²

##  Models Used
- Linear Regression
- K-Nearest Neighbors (KNN)

## Files
- `house_price.ipynb`
- `house_price.csv`

##  Result
Linear Regression performed better compared to KNN for this dataset.
